# CMC Visuals — Subject Media Creator Storefront

A reusable, config-driven storefront + backend for SubjectSkillz creators.
One creator per config file (or per onboarding flow). Part of the Trail of Joy
ecosystem. See DEPLOY.md to run and test, and trail-of-joy SOPs for the system.

Stack: Next.js (app-router) · Supabase (multi-tenant, RLS) · Stripe Connect ·
HighLevel community bridge · Resend email.

Automation target: 95% of mechanical work. Interpersonal steps (the two calls,
cohorts, 1-on-1s) are intentionally human.
